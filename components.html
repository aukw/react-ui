General

ReactUI uses Statefy to control state and props. We recommend reading it's documentation.

Auto generated events

Statefy adds auto-generated change events to any props. So, for instance, if a component has a url prop, then, there's also a onUrlChange event. When this event is handled, ifevent.preventDefault() is called, then the component will not update it's internal state and it's up the the caller to render the component again.
